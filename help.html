SimViz

SimViz is a visualization tool for exploring case bases and similarity functions.

Use the dropdown lists on the left to select a dataset and a similarity function.
Click on the heatmap for more details on the similarity between two cases
Click on the histogram to randomly select a pair of cases according to their similarity value
Click on the pin icon next to the case id to review the similarity of this case with the rest of the case base
Click on button to recompute a similarity function modifying the Local similarity weights
